# Basic mathematical concepts

These are the mathematical concepts used in the documentation. It is far from complete.

---
<a id="mathematical-object"></a>
- **object** — a unique thing

---
<a id="natural-number"></a>
- **natural number**
    - formulation: $\mathbb{N} = \{1,2,\dots\}$
    - successor: $n \mapsto n+1$


---
<a id="pair"></a>
- **pair** — ordered pair of two objects
    - first element: object $x$
    - second element: object $y$
    - formulation: $(x,y)$

---

<a id="mathematical-set"></a>
- **set** — collection of distinct objects
    - notation: $X$
    - formulation: $\{x_1, x_2, \dots\}$
    - element: an object contained in the set
    - cardinality (size): $|X| \in \mathbb{N} \cup \{\infty\}$
    - subset: a set whose elements are all contained in $X$

---

<a id="list"></a>
- **list** ↩  [set](#mathematical-set)
    - formulation: $(x_1, x_2, \dots)$
    - sublist: a list obtained by removing elements while preserving order
    - next: gives the successor element
    - note: oreder set

---

- **element relation** ($\in$)
    - type: binary relation (predicate)
    - arguments: object $x$, set $X$
    - value:
        - true if $x$ is an element of $X$
        - false otherwise
    - formulation:
      $x \in X$

---

- **subset relation** ($\subseteq$)
    - type: binary relation (predicate)
    - arguments: sets $Y$, $X$
    - value:
        - true if every element of $Y$ is an element of $X$
    - formulation:
      $Y \subseteq X$

---

- **superset relation** ($\supseteq$)
    - type: binary relation (predicate)
    - arguments: sets $X$, $Y$
    - value:
        - true if every element of $Y$ is an element of $X$
    - formulation:
      $X \supseteq Y$

---
<a id="cartesian-product"></a>
- **Cartesian product** ↩ [set](#mathematical-set)
    - operands: sets $X$, $Y$
    - definition:
      the set of all ordered pairs with first element from $X$ and second from $Y$
    - formulation:
      $X \times Y = \{ (x,y) \mid x \in X,\; y \in Y \}$
    - element:
      ordered pair $(x,y)$

---

<a id="map"></a>
- **map** (relation) ↩ subset of [Cartesian product](#cartesian-product)
    - definition:
      a subset $M \subseteq X \times Y$
    - note:
      a map may associate zero, one, or multiple elements of $Y$ to an element of $X$
    - domain:
      $\operatorname{Dom}(M)
      = \{ x \in X \mid \exists y \in Y:\; (x,y) \in M \}$
    - range:
      $\operatorname{Ran}(M)
      = \{ y \in Y \mid \exists x \in X:\; (x,y) \in M \}$
    - codomain:
      the set $Y$ specified in $X \times Y$

---
<a id="image"></a>
- **image** (fiber of a map)
    - type:
      set-valued operation induced by a map
    - arguments:
        - map $M \subseteq X \times Y$
        - subset $A \subseteq X$
          (or $A=\{x\}$ for an object $x$)
    - value:
      the set of all elements of $Y$ that are related to at least one element of $A$
    - formulation:
      $M(A)
      = \{\, y \in Y \mid \exists x \in A:\; (x,y) \in M \,\}$
    - special case ($A=\{x\}$):
        - image of $x$
        - fiber over $x$
    - note:
      generalizes the image of a single element

---
<a id="inverse-image"></a>
- **inverse image** (preimage)
    - type:
      set-valued operation induced by a map
    - arguments:
        - map $M \subseteq X \times Y$
        - subset $B \subseteq Y$
          (or $B=\{y\}$ for an object $y$)
    - value:
      the set of all elements of $X$ that are related to at least one element of $B$
    - formulation:
      $M^{-1}(B)
      = \{\, x \in X \mid \exists y \in B:\; (x,y) \in M \,\}$

---
<a id="total-map"></a>
- **total map**: ↩ [map](#map)
    - condition:
      $\operatorname{Dom}(M) = X$

---
<a id="function"></a>
- **function** (functional map):  ↩ [map](#map)
    - conditions:
        - $\operatorname{Dom}(M) = X$
        - $\forall x \in X:\; |M(x)| = 1$
    - notation: $f : X \to Y$
    
---
<a id="partition-of-a-set"></a>
- **partition**
    - type:
      set-valued structure on a set
    - arguments:
        - underlying set $X$
    - value:
      a set $C$ of subsets of $X$
    - elements:
      subsets $c \subseteq X$
    - completeness condition:
      $\displaystyle \bigcup_{c \in C} c = X$
    - disjointness condition:
      $c \cap c' = \emptyset
      \quad \text{for } c \neq c',\; c,c' \in C$
    - interpretation:
      each element of $X$ belongs to exactly one element of $C$
    - induced structure:
      σ-algebra generated by the partition

---
<a id="sigma-algebra"></a>
- **σ-algebra**
    - type:
      collection of subsets of a set
    - arguments:
        - underlying set $X$
        - generating partition $C$
    - value:
      a set $\sigma(C)$ of subsets of $X$
    - definition:
      $\displaystyle
      \sigma(C)
      = \left\{
          \bigcup_{i \in J} c_i
          \;\middle|\;
          J \subseteq I
        \right\},
      \quad C=\{c_i\}_{i\in I}$
    - properties:
        - $\emptyset \in \sigma(C)$
        - $X \in \sigma(C)$
        - closed under complement
        - closed under countable unions
    - note:
      smallest σ-algebra containing all elements of the partition
